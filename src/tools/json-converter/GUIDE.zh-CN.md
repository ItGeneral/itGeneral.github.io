# JSON 转换器使用指南

## 快速开始

### 1. 输入 JSON

在左侧输入框中粘贴或输入 JSON 数据：

```json
{
  "name": "张三",
  "age": 25,
  "email": "zhangsan@example.com"
}
```

### 2. 选择转换格式

点击顶部的格式按钮选择输出格式：

- **格式化**：美化 JSON，添加缩进和换行
- **压缩**：移除所有空格和换行
- **YAML**：转换为 YAML 格式
- **XML**：转换为 XML 格式
- **CSV**：转换为 CSV 格式
- **TypeScript**：生成 TypeScript interface
- **Java**：生成 Java 类
- **Go**：生成 Go 结构体
- **Schema**：生成 JSON Schema

### 3. 查看和复制结果

右侧自动显示转换结果，点击 **📋 复制** 按钮复制结果。

## 转换示例

### JSON → TypeScript

**输入：**
```json
{
  "id": 1,
  "name": "产品A",
  "price": 99.99,
  "inStock": true
}
```

**输出：**
```typescript
interface Product {
  id: number;
  name: string;
  price: number;
  inStock: boolean;
}
```

### JSON → Go

**输入：**
```json
{
  "user_id": 123,
  "user_name": "John",
  "is_active": true
}
```

**输出：**
```go
type AutoGenerated struct {
    UserId   int    `json:"user_id"`
    UserName string `json:"user_name"`
    IsActive bool   `json:"is_active"`
}
```

### JSON → CSV

**输入：**
```json
[
  {"name": "张三", "age": 25},
  {"name": "李四", "age": 30}
]
```

**输出：**
```csv
name,age
张三,25
李四,30
```

## JSON Path 查询

使用 JSONPath 表达式查询 JSON 数据：

**示例数据：**
```json
{
  "store": {
    "book": [
      {"category": "reference", "author": "Nigel Rees", "price": 8.95},
      {"category": "fiction", "author": "Evelyn Waugh", "price": 12.99}
    ]
  }
}
```

**查询表达式：**
- `$.store.book[*].author` - 获取所有书籍作者
- `$..book[?(@.price < 10)]` - 获取价格小于 10 的书
- `$.store.*` - 获取 store 下所有内容

## 常见问题

**Q: JSON 验证失败？**
A: 检查：
- 键名必须使用双引号
- 最后一个元素后不能有逗号
- 字符串必须使用双引号
- 检查括号是否匹配

**Q: 转换为 CSV 失败？**
A: CSV 转换要求：
- 输入必须是数组格式
- 数组元素必须是对象
- 所有对象的键应该一致

**Q: Go 字段名不对？**
A: Go 转换器会：
- 自动将 JSON 字段名转换为 Go 命名（驼峰）
- 首字母大写（导出字段）
- 保留 `json` 标签
