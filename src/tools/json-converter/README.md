# JSON 转换器 / JSON Converter

JSON 转换器是一个强大的 JSON 数据处理工具，支持格式化、压缩以及多种格式的相互转换。

## 功能特点

- ✨ **实时验证**：实时检查 JSON 语法错误
- 🎨 **语法高亮**：美化 JSON 显示，提升可读性
- 🔄 **多格式转换**：支持 JSON、YAML、XML、CSV、TypeScript、Java、Go 等格式
- 📊 **数据统计**：实时显示字符数、行数等信息
- 📋 **一键复制**：快速复制转换结果
- 🔍 **JSON Path 查询**：支持 JSONPath 表达式查询

## 支持的转换格式

### 输入格式
- **JSON**：标准 JSON 格式

### 输出格式
1. **JSON 格式化**：美化 JSON，添加缩进和换行
2. **JSON 压缩**：移除所有空格和换行
3. **YAML**：转换为 YAML 格式
4. **XML**：转换为 XML 格式
5. **CSV**：转换为 CSV 格式
6. **TypeScript**：生成 TypeScript interface
7. **Java**：生成 Java 类
8. **Go**：生成 Go 结构体
9. **JSON Schema**：生成 JSON Schema
10. **JSON Path**：支持 Path 查询

## 使用方法

### 1. 输入 JSON

在左侧输入框中粘贴或输入 JSON 数据：

```json
{
  "name": "张三",
  "age": 25,
  "email": "zhangsan@example.com",
  "skills": ["JavaScript", "Python", "Go"]
}
```

### 2. 选择转换格式

点击顶部的格式按钮选择输出格式：
- **格式化**：美化 JSON
- **压缩**：压缩 JSON
- **YAML**：转换为 YAML
- **XML**：转换为 XML
- **CSV**：转换为 CSV
- **TypeScript**：生成 TypeScript
- **Java**：生成 Java 类
- **Go**：生成 Go 结构体
- **Schema**：生成 JSON Schema

### 3. 查看结果

右侧会自动显示转换结果。

### 4. 复制结果

点击 **📋 复制** 按钮复制结果到剪贴板。

## 转换示例

### JSON → YAML

**输入：**
```json
{
  "name": "张三",
  "age": 25
}
```

**输出：**
```yaml
name: 张三
age: 25
```

### JSON → TypeScript Interface

**输入：**
```json
{
  "id": 1,
  "name": "产品A",
  "price": 99.99,
  "inStock": true
}
```

**输出：**
```typescript
interface Product {
  id: number;
  name: string;
  price: number;
  inStock: boolean;
}
```

### JSON → Go Struct

**输入：**
```json
{
  "user_id": 123,
  "user_name": "John",
  "is_active": true
}
```

**输出：**
```go
type AutoGenerated struct {
    UserId   int     `json:"user_id"`
    UserName string  `json:"user_name"`
    IsActive bool    `json:"is_active"`
}
```

### JSON → CSV

**输入：**
```json
[
  {"name": "张三", "age": 25},
  {"name": "李四", "age": 30}
]
```

**输出：**
```csv
name,age
张三,25
李四,30
```

## JSON Path 查询

支持使用 JSONPath 表达式查询 JSON 数据：

### 示例

**数据：**
```json
{
  "store": {
    "book": [
      {"category": "reference", "author": "Nigel Rees", "price": 8.95},
      {"category": "fiction", "author": "Evelyn Waugh", "price": 12.99}
    ]
  }
}
```

**查询表达式：**
- `$.store.book[*].author` - 获取所有书籍作者
- `$..book[?(@.price < 10)]` - 获取价格小于 10 的书
- `$.store.*` - 获取 store 下所有内容

## 常见问题

### Q: JSON 验证失败？
A: 检查以下几点：
- 键名必须使用双引号
- 最后一个元素后不能有逗号
- 字符串必须使用双引号（不能单引号）
- 检查括号是否匹配

### Q: 转换为 CSV 失败？
A: CSV 转换要求数据格式：
- 输入必须是数组格式
- 数组元素必须是对象
- 所有对象的键应该一致

### Q: Go 字段名不对？
A: Go 转换器会：
- 自动将 JSON 字段名转换为 Go 命名（驼峰）
- 首字母大写（导出字段）
- 保留 `json` 标签

### Q: 大 JSON 处理慢？
A: 建议：
- 先压缩再转换
- 分批处理大文件
- 使用 JSON Path 提取需要的部分

## 性能建议

- 小于 1MB：实时处理无压力
- 1MB - 10MB：可能需要等待几秒
- 大于 10MB：建议先压缩或分块处理
