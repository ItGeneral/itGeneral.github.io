# JSON Converter User Guide

## Quick Start

### 1. Input JSON

Paste or type JSON data in the left input box:

```json
{
  "name": "John",
  "age": 25,
  "email": "john@example.com"
}
```

### 2. Select Conversion Format

Click format buttons at the top to select output format:

- **Format** - Beautify JSON with indentation
- **Minify** - Remove all spaces and line breaks
- **YAML** - Convert to YAML format
- **XML** - Convert to XML format
- **CSV** - Convert to CSV format
- **TypeScript** - Generate TypeScript interface
- **Java** - Generate Java class
- **Go** - Generate Go struct
- **Schema** - Generate JSON Schema

### 3. View and Copy Result

Result automatically appears on the right. Click **📋 Copy** button to copy.

## Conversion Examples

### JSON → TypeScript

**Input:**
```json
{
  "id": 1,
  "name": "Product A",
  "price": 99.99,
  "inStock": true
}
```

**Output:**
```typescript
interface Product {
  id: number;
  name: string;
  price: number;
  inStock: boolean;
}
```

### JSON → Go

**Input:**
```json
{
  "user_id": 123,
  "user_name": "John",
  "is_active": true
}
```

**Output:**
```go
type AutoGenerated struct {
    UserId   int    `json:"user_id"`
    UserName string `json:"user_name"`
    IsActive bool   `json:"is_active"`
}
```

### JSON → CSV

**Input:**
```json
[
  {"name": "John", "age": 25},
  {"name": "Jane", "age": 30}
]
```

**Output:**
```csv
name,age
John,25
Jane,30
```

## JSON Path Query

Use JSONPath expressions to query JSON data:

**Sample Data:**
```json
{
  "store": {
    "book": [
      {"category": "reference", "author": "Nigel Rees", "price": 8.95},
      {"category": "fiction", "author": "Evelyn Waugh", "price": 12.99}
    ]
  }
}
```

**Query Expressions:**
- `$.store.book[*].author` - Get all book authors
- `$..book[?(@.price < 10)]` - Get books under $10
- `$.store.*` - Get everything under store

## FAQ

**Q: JSON validation failed?**
A: Check:
- Key names must use double quotes
- No trailing comma after last element
- Strings must use double quotes
- Check if brackets are matched

**Q: CSV conversion failed?**
A: CSV conversion requires:
- Input must be in array format
- Array elements must be objects
- All objects should have consistent keys

**Q: Go field names incorrect?**
A: Go converter will:
- Auto-convert JSON field names to Go naming (camelCase)
- Capitalize first letter (exported fields)
- Keep `json` tags
