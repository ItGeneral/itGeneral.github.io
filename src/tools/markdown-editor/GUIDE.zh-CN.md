# Markdown 编辑器使用指南

## 快速开始

### 1. 创建文档

点击左侧边栏的 **+ Markdown Docs** 按钮，自动创建一个新文档。

### 2. 编辑文档

在左侧编辑区输入 Markdown 内容，右侧预览区实时显示渲染结果。

### 3. 自动保存

编辑内容自动保存到本地浏览器。

## Markdown 基础语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式

```markdown
**粗体文本**
*斜体文本*
~~删除线文本==
`行内代码`
```

### 列表

**无序列表：**
```markdown
- 项目 1
- 项目 2
  - 子项目
```

**有序列表：**
```markdown
1. 第一项
2. 第二项
```

### 链接和图片

```markdown
[链接文本](https://example.com)
![图片描述](https://example.com/image.jpg)
```

### 引用

```markdown
> 这是一段引用文本
>
> 可以多行
```

### 代码块

````markdown
```javascript
function hello() {
  console.log('Hello!');
}
```
````

### 表格

```markdown
| 列 1 | 列 2 |
|------|------|
| 数据 1 | 数据 2 |
```

### 任务列表

```markdown
- [x] 已完成任务
- [ ] 未完成任务
```

## 管理文档

### 新建文档
点击 **+ Markdown Docs** 按钮

### 切换文档
点击左侧边栏的文档项

### 重命名文档
1. 鼠标悬停在文档项上
2. 点击 **✎** 图标
3. 输入新名称
4. 按 Enter 确认

### 删除文档
1. 鼠标悬停在文档项上
2. 点击 **×** 图标
3. 确认删除

## 查看统计信息

底部状态栏显示：
- **字数**：文档总字数
- **字符**：总字符数
- **行数**：总行数
- **阅读时间**：预计阅读时间

## 使用技巧

### 快速排版

#### 引用代码块
输入 ``` ` ``` 后直接输入语言名称，如：
```javascript
```

#### 快速创建列表
输入 `- ` 或 `1. ` 后按空格，自动创建列表。

#### 快速创建标题
输入 `# ` 后按空格，自动创建标题。

### 嵌套列表

使用 Tab 键创建嵌套：
```markdown
- 一级项目
  - 二级项目
    - 三级项目
```

### 转义字符

使用 `\` 转义特殊字符：
```markdown
\* 不是斜体
\[ 不是链接
```

## 注意事项

⚠️ **数据安全**
- 文档保存在浏览器本地存储（LocalStorage）
- 清除浏览器数据会丢失文档
- 建议定期导出重要文档备份

⚠️ **兼容性**
- 支持 CommonMark 标准
- 支持 GitHub Flavored Markdown (GFM)
- 部分高级特性可能不完全支持

## 常见问题

**Q: 编辑和预览不同步？**
A: 检查：
- Markdown 语法是否正确
- 是否有不匹配的括号
- 特殊字符是否需要转义

**Q: 代码高亮不生效？**
A: 确保代码块语言标识正确：
- 使用 `javascript` 而非 `js`
- 使用 `python` 而非 `py`
- 语言名称小写

**Q: 表格显示错乱？**
A: 检查表格格式：
- 列分隔符 `|` 必须对齐
- 表头和内容间必须有 `|------|` 分隔
- 每行列数必须一致

**Q: 文档丢失？**
A: 可能原因：
- 清除了浏览器缓存
- 使用了隐私/无痕模式
- 浏览器限制存储空间

**预防措施：**
- 定期导出文档
- 使用多个浏览器
- 启用浏览器同步
