# 文本去重 / Text Dedup

文本去重工具是一个简单实用的文本行去重工具，帮助您快速清理重复内容。

## 功能特点

- 🎯 **快速去重**：一键去除重复行
- 🔧 **自定义分隔符**：支持换行、逗号、空格、制表符、分号等
- ⚙️ **灵活选项**：忽略大小写、去除空格、排序等
- 📊 **实时统计**：显示原始条数、去重后数量、删除数量
- 🎨 **重复项高亮**：高亮显示被删除的重复项
- 📋 **一键复制**：快速复制去重结果

## 使用方法

### 1. 输入文本

在左侧输入框中输入需要去重的文本，每行一条：

```
apple
banana
Apple
cherry
banana
date
cherry
elderberry
APPLE
fig
grape
```

### 2. 选择分隔符

在顶部工具栏选择分隔符：
- **换行**：每行一条（默认）
- **逗号**：逗号分隔
- **空格**：空格分隔
- **制表符**：Tab 分隔
- **分号**：分号分隔
- **自定义**：输入自定义分隔符

### 3. 配置选项

勾选需要的选项：
- **忽略大小写**：`Apple` 和 `apple` 视为相同
- **去除首尾空格**：自动去除每行首尾空格
- **去除空项**：自动删除空行
- **排序结果**：去重后按字母顺序排序

### 4. 查看结果

右侧自动显示去重后的结果，底部显示：
- **原始条数**：输入的总行数
- **去重后**：去重后的行数
- **删除重复**：删除的重复项数量

### 5. 复制结果

点击 **📋 复制** 按钮复制结果。

## 选项详解

### 忽略大小写

勾选后，大小写不同的单词被视为相同：
- `Apple` = `apple` = `APPLE`
- `Hello` = `hello` = `HELLO`

**示例：**
```
输入：Apple, apple, APPLE
不忽略大小写：Apple, apple, APPLE（3 项）
忽略大小写：Apple（1 项）
```

### 去除首尾空格

自动去除每行首尾的空格和制表符：
- `  张三  ` → `张三`
- `  李四` → `李四`
- `王五  ` → `王五`

### 去除空项

自动删除空行或只包含空格的行：
- ``（空行）→ 删除
- `   `（只有空格）→ 删除
- `text`（有内容）→ 保留

### 排序结果

去重后按字母顺序排序：
- 数字排序：1, 2, 10, 100
- 字母排序：Apple, Banana, Cherry
- 中文排序：按拼音排序

**注意：** 排序会影响原始顺序，如需保留顺序请勿勾选。

## 常见问题

### Q: 去重不完整？
A: 检查以下几点：
- 分隔符是否正确
- 是否勾选了"忽略大小写"
- 是否有隐藏字符（空格、制表符）
- 尝试勾选"去除首尾空格"

### Q: 排序顺序不对？
A: 中文排序按拼音顺序：
- 一、二、三
- a、b、c

### Q: 自定义分隔符不生效？
A: 注意：
- 自定义分隔符是精确匹配
- 注意空格：` ` 和 `  ` 不同
- 特殊字符需要正确输入：`\t` 表示制表符

### Q: 大文本处理慢？
A: 建议：
- 分批处理
- 关闭不需要的选项
- 先清空再输入新内容

## 注意事项

⚠️ **数据限制**
- 建议单次处理不超过 100,000 行
- 超大数据量可能导致浏览器卡顿

⚠️ **内存使用**
- 大文本会占用较多内存
- 处理完成后建议清空数据

⚠️ **数据安全**
- 所有处理在本地浏览器完成
- 数据不会上传到服务器
