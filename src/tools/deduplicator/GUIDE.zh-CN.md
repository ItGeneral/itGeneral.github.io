# 文本去重使用指南

## 快速开始

### 1. 输入文本

在左侧输入框中输入需要去重的文本，每行一条：

```
apple
banana
Apple
cherry
banana
date
```

### 2. 选择分隔符

在顶部工具栏选择分隔符：
- **换行**：每行一条（默认）
- **逗号**：逗号分隔
- **空格**：空格分隔
- **制表符**：Tab 分隔
- **分号**：分号分隔
- **自定义**：输入自定义分隔符

### 3. 配置选项

勾选需要的选项：
- **忽略大小写**：`Apple` 和 `apple` 视为相同
- **去除首尾空格**：自动去除每行首尾空格
- **去除空项**：自动删除空行
- **排序结果**：去重后按字母顺序排序

### 4. 查看和复制结果

右侧自动显示去重后的结果，点击 **📋 复制** 按钮复制结果。

## 使用场景

### 清理重复邮箱

**输入：**
```
user1@example.com
user2@example.com
user1@example.com
user3@example.com
```

**设置：**
- 分隔符：换行
- 去除空项：✓

**输出：**
```
user1@example.com
user2@example.com
user3@example.com
```

### 去重关键词

**输入：**
```
JavaScript, Python, Go, JavaScript, Java, Python
```

**设置：**
- 分隔符：逗号
- 排序结果：✓

**输出：**
```
Go, Java, JavaScript, Python
```

### 忽略大小写去重

**输入：**
```
APPLE
banana
APPLE
Cherry
```

**设置：**
- 分隔符：换行
- 忽略大小写：✓

**输出：**
```

```

## 选项详解

### 忽略大小写

勾选后，大小写不同的单词被视为相同：
- `Apple` = `apple` = `APPLE`
- `Hello` = `hello` = `HELLO`

### 去除首尾空格

自动去除每行首尾的空格和制表符：
- `  张三  ` → `张三`
- `  李四` → `李四`

### 去除空项

自动删除空行或只包含空格的行：
- ``（空行）→ 删除
- `   `（只有空格）→ 删除

### 排序结果

去重后按字母顺序排序：
- 数字：1, 2, 10, 100
- 字母：Apple, Banana, Cherry
- 中文：按拼音排序

## 使用技巧

### 处理 CSV 数据

**输入：**
```
name,email,age
张三,zhangsan@example.com,25
张三,zhangsan@example.com,25
```

**设置：**
- 分隔符：换行
- 去除空项：✓

**输出：**
```
name,email,age
张三,zhangsan@example.com,25
```

### 处理标签

**输入：**
```
#JavaScript #Python #Go #JavaScript
```

**设置：**
- 分隔符：自定义
- 自定义分隔符：` #`（空格 + #）
- 去除首尾空格：✓

**输出：**
```
#Go #JavaScript #Python
```

## 常见问题

**Q: 去重不完整？**
A: 检查：
- 分隔符是否正确
- 是否勾选了"忽略大小写"
- 是否有隐藏字符（空格、制表符）
- 尝试勾选"去除首尾空格"

**Q: 自定义分隔符不生效？**
A: 注意：
- 自定义分隔符是精确匹配
- 注意空格：` ` 和 `  ` 不同
- 特殊字符需要正确输入

**Q: 大文本处理慢？**
A: 建议：
- 分批处理
- 关闭不需要的选项
- 建议单次处理不超过 100,000 行

## 注意事项

⚠️ **数据限制**
- 建议单次处理不超过 100,000 行
- 超大数据量可能导致浏览器卡顿

⚠️ **数据安全**
- 所有处理在本地浏览器完成
- 数据不会上传到服务器
