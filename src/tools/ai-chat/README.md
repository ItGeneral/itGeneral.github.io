# AI 助手 / AI Assistant

AI 助手是一个强大的 AI 对话工具，支持多种 AI 提供商，让您可以与 AI 进行实时对话、代码调试、问题解答等。

## 功能特点

- 🤖 **多 AI 提供商支持**：支持 OpenAI、Anthropic、阿里云等多种 AI 服务
- 💬 **对话管理**：创建多个对话，支持重命名和删除
- 📎 **文件上传**：支持上传文本文件作为上下文
- 🔄 **实时流式响应**：实时显示 AI 回复内容
- 📝 **Markdown 支持**：完美支持 Markdown 格式渲染
- 🌍 **多语言界面**：支持中文和英文界面切换

## 使用方法

### 1. 配置 AI

首次使用需要配置 AI 服务：

1. 点击右上角的 **⚙ 设置** 按钮
2. 选择 AI 提供商（OpenAI、Anthropic 等）
3. 输入 API Key
4. 设置 API 地址（可选）
5. 选择模型（如 gpt-4o-mini）
6. 点击 **测试并保存**

### 2. 开始对话

1. 在输入框中输入您的问题
2. 按 **Enter** 发送消息
3. 按 **Shift + Enter** 换行
4. AI 会实时回复您的消息

### 3. 管理对话

- **新建对话**：点击左侧边栏的 **+ 新建对话** 按钮
- **切换对话**：点击左侧边栏中的对话项
- **重命名对话**：鼠标悬停在对话项上，点击 **✎** 图标
- **删除对话**：鼠标悬停在对话项上，点击 **×** 图标

### 4. 上传文件

1. 点击输入框右侧的 **📎** 图标
2. 选择要上传的文本文件
3. 文件内容会自动添加到消息中

或直接拖拽文件到聊天区域。

### 5. 复制回复

鼠标悬停在 AI 的回复上，点击 **📋** 图标复制内容。

## 快捷操作

- **Enter**：发送消息
- **Shift + Enter**：换行
- **拖拽文件**：上传文件作为上下文

## 支持的 AI 提供商

- **API 地址**：`https://api.openai.com/v1`
- **推荐模型**：
  - `gpt-4o-mini`：快速且经济
  - `gpt-4o`：性能强大
  - `gpt-3.5-turbo`：性价比高

- **推荐模型**：
  - `claude-3-5-sonnet-20241022`：最新版本

## 使用建议

### 1. 编程相关
- **调试代码**：直接粘贴代码和错误信息，让 AI 帮助分析
- **代码解释**：选中不懂的代码片段，询问 AI
- **代码优化**：让 AI 帮助优化代码性能

### 2. 文本处理
- **文本摘要**：粘贴长文本，让 AI 提供摘要
- **内容翻译**：让 AI 翻译不同语言的内容
- **风格改写**：让 AI 改写文本风格

### 3. 学习辅助
- **概念解释**：询问技术概念的详细解释
- **最佳实践**：询问行业最佳实践
- **问题排查**：描述问题，让 AI 提供排查思路

## 注意事项

⚠️ **API Key 安全**
- API Key 仅保存在本地浏览器中
- 不会上传到任何服务器
- 请勿分享您的 API Key

⚠️ **文件限制**
- 单个文件大小限制：100KB
- 超过限制会提示确认
- 大文件可能超出 AI 上下文限制

⚠️ **网络要求**
- 需要稳定的网络连接
- 某些地区可能需要科学上网

## 常见问题

### Q: AI 回复很慢？
A: 检查网络连接，或切换到更快的模型（如 gpt-4o-mini）。

### Q: 提示 "请先配置 AI"？
A: 点击设置按钮，配置 AI 提供商和 API Key。

### Q: 上传文件失败？
A: 确保文件是文本格式，且大小不超过 100KB。

### Q: 如何查看对话历史？
A: 所有对话都保存在左侧边栏，点击即可切换。
