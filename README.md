## ✨ Features

- **🔒 Privacy-First Design**: All data stays in your browser. No backend servers, no data collection.
- **⚡ Fast & Reliable**: Everything runs locally with instant response times.
- **🎨 Beautiful Interface**: Clean, modern design with dark mode support.
- **🌍 Multi-Language**: Supports English and Chinese (中文).
- **📱 Responsive**: Works perfectly on desktop, tablet, and mobile devices.
- **🆓 Completely Free**: Open source and free to use.

## 🛠️ Tools

### 📝 Markdown Editor
A powerful distraction-free writing environment with live preview and syntax highlighting.

**Features:**
- Real-time markdown preview
- Syntax highlighting for code blocks
- Auto-save to browser storage
- Toolbar with formatting options
- Word count and reading time
- Export to HTML/Markdown

**Use Cases:**
- Writing technical documentation
- Creating README files
- Drafting blog posts
- Taking notes

### 🔄 JSON Converter
A versatile JSON processing tool for formatting, validating, and converting JSON data.

**Features:**
- Format and beautify JSON
- Minify JSON for production
- Validate JSON syntax
- Convert to YAML, XML, CSV
- Generate TypeScript interfaces
- JSON Schema generation
- JSONPath query support

**Use Cases:**
- Debugging API responses
- Converting configuration files
- Generating TypeScript interfaces
- Cleaning up JSON data

### 🎯 Regex Tester
An advanced regular expression testing tool with visual feedback and railroad diagrams.

**Features:**
- Real-time matching highlight
- Capture groups analysis
- Visual railroad diagrams
- Common patterns library
- Match information display
- Multiple string testing

**Use Cases:**
- Creating validation patterns
- Parsing text data
- Replacing text with complex rules
- Learning regex

### 🧹 Deduplicator
A simple yet powerful tool for removing duplicate lines from text lists.

**Features:**
- Instant duplicate removal
- Case-sensitive option
- Trim whitespace
- Sort output
- One-click copy to clipboard

**Use Cases:**
- Cleaning email lists
- Removing duplicate URLs
- Organizing keyword lists
- Data cleanup tasks

### 🤖 AI Assistant
A conversational AI interface connecting to leading AI providers for coding, writing, and analysis.

**Features:**
- Multiple AI providers (OpenAI, Anthropic, DeepSeek)
- Bring your own API key
- Conversation history saved locally
- File attachments support
- Markdown rendering
- Multiple conversations

**Use Cases:**
- Getting coding help
- Debugging code
- Writing documentation
- Generating code snippets
- Learning programming languages

## 🚀 Getting Started

### Online Demo

Visit: **[https://itgeneral.github.io/ai-tools/](https://itgeneral.github.io/ai-tools/)**

No installation required. Just open the link and start using!

### Local Development

1. **Clone the repository**
```bash
git clone https://github.com/ItGeneral/ai-tools.git
cd ai-tools
```

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

4. **Open your browser**
Navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## 🏗️ Technology Stack

- **Framework**: Vue 3 + TypeScript
- **Build Tool**: Vite
- **Routing**: Vue Router 4 (Hash Mode)
- **Editor**: CodeMirror 6
- **Markdown**: Marked + DOMPurify
- **Styling**: Custom CSS with scoped styles
- **AI Integration**: OpenAI, Anthropic, DeepSeek APIs

## 🔒 Privacy & Security

### Data Storage
- **All data is stored locally** in your browser's localStorage
- No backend servers collect or store your information
- Your documents, conversations, and settings stay on your device
- You have complete control over your data

### AI Chat Feature
- Messages are sent **directly** to AI providers you configure
- We never see or store your API keys or conversations
- API keys are stored only in your browser's localStorage
- Your use of AI providers is governed by their respective privacy policies

### Third-Party Services
- **Google AdSense**: Used for displaying ads (uses cookies)
- **Analytics**: Anonymous usage statistics to improve performance

For detailed information, please review our [Privacy Policy](https://itgeneral.github.io/ai-tools/page/privacy) and [Terms of Service](https://itgeneral.github.io/ai-tools/page/terms).
